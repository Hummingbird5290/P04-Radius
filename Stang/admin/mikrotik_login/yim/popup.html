การตั้งค่าให้เครื่อง computerของเราแสดง pop-up สเตตัสการใช้งาน

1.       ตรวจสอบเครื่อง computer ของเราก่อนว่าได้รับไอพีอยู่ในกรุ๊ปไหน โดยเปิด web browser ที่เราใช้ประจำขึ้นมา จะพบกับหน้าตาของหน้า login เข้าใช้งานระบบ hotspot

จดเลข IP ตามที่ได้เหมือนรูปด้านบนจะเป็น IP 10.0.13.1  

2.       เข้าไปตั้งค่าใน web browser  มีวิธีดังต่อไปนี้

2.1            Internet Explorer (IE)

       เข้าไปที่เมนู Tools ,Pop-up Blocker ,Pop-up Blocker Settings

ใส่ IP ที่เราจดไว้ในข้อที่ 1  จากนั้นกดปุ่ม Add , Close  เป็นอันเรียบร้อยสำหรับ IE

2.2            Google Chrome

       จากหน้าแรกมองหาไอคอนรูปประแจ มุมขวาบนกดตรงไอคอนดังกล่าวแล้วเข้าไปยังเมนูต่อไปนี้

      Options , Under the Hood , Privacy , Content Settings , Pop-ups ,Manage Exceptions

ใส่ IP ที่เราจดไว้ในข้อที่ 1  จากนั้นกดปุ่ม Enter เป็นอันเรียบร้อยสำหรับ Google Chrome

 

2.3            Mozilla Firefox

มองหาเมนู Firefox สีส้มโดยค่า default แล้วจะอยู่ตรงมุมซ้ายบนของ web browser จากนั้นเข้าไปยังเมนูต่อไปนี้

Options , Options , เลือก Tab Content ,ตรง Block Pop-up windows กดปุ่ม Exceptions

ใส่ IP ที่เราจดไว้ในข้อที่ 1  จากนั้นกดปุ่ม Allow ,Close เป็นอันเรียบร้อยสำหรับ Firefox
